This version treats each saved row as an evaluated archive rather than a self-description by a collaborator. The goal is comparative placement, not matchmaking.

What the sliders mean

Each slider represents one epiaesthetic continuum. A score of -10 places the archive strongly toward Pole A, 0 marks an ambivalent or mixed reading, and +10 places it toward Pole B.

How to use the tool

Enter an archive, collection, platform, repository, or media object, then score it across all active continuums. Once saved, the archive appears in the field and becomes available for comparative distance analysis.

Editing mode

Use the “Load Existing Archive” dropdown to select a saved archive. The form will load its saved values and updates will be written back to the same record by Archive ID.

Suggested continuums

Time bias / Space bias, Singular / Multiple, Closed / Open, Vertical / Horizontal, Passive / Active, High causality / Low causality, and Hegemonic / Counter-hegemonic.

Overall Distance Matrix

Lighter cells indicate archives that are closer together across the full set of continuums. Darker cells indicate greater distance.
